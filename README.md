📌 ReadMe: Lab2 Encryption Tool

📂 Содержимое репозитория

💻 Исходники

🔹 Lab2.exe

Запускаемый файл программы. Используйте его для шифрования и расшифровки файлов.

📝 test-table.xlsx

Файл с тестовыми данными для проверки работы программы.

📸 screenshots.docx

Файл с изображениями, демонстрирующими работу программы.

📁 testfile

Папка с тестовыми файлами для проверки алгоритма шифрования и дешифрования.

📁 result

Папка с результатами работы программы:

Файлы с расширением .encrypto – зашифрованные версии исходных файлов.

Файлы с расширением .dectypto – расшифрованные версии зашифрованных файлов.

🚀 Как использовать

1. Запустите Lab2.exe.

2. Выберите файл для шифрования/дешифрования

3. Сохраните результат
